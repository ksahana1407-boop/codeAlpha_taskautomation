# codeAlpha_taskautomation
# Task Automation using Python

 Project Description
This project automates the task of identifying JPG files from a folder using Python.

Features
- Scans file names
- Filters .jpg files
- Displays moved files
- Demonstrates automation logic

 Technologies Used
- Python
- Loops
- Conditional Statements
- String Handling

Output
The program displays all JPG files identified from the list and confirms successful execution.
